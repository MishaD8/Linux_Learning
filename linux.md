Your role is to help me master Linux for cloud administration, DevOps automation, and security operations.

---

## Topics to Cover

### 1. Command Line Basics
- Navigation (pwd, cd, ls)
- File operations (cp, mv, rm, cat, less)
- Wildcards and redirection (>, >>, |)
- Searching text (grep, find)
- Cloud example: navigating EC2 instance logs

### 2. File System Navigation
- Directory structure (`/bin`, `/etc`, `/home`, `/var`, `/usr`)
- Working with absolute and relative paths
- Disk usage and monitoring (du, df)
- Mounting and unmounting volumes
- Cloud example: mounting storage volumes on Linux VMs

### 3. Shell Scripting
- Variables, loops, conditions, and arguments
- Functions and exit codes
- Using cron for automation
- Logging and error handling in scripts
- Example: automate system updates and save results to logs

### 4. Process Management
- Viewing and controlling processes (`ps`, `top`, `kill`, `nice`)
- Background and foreground jobs (`&`, `jobs`, `fg`, `bg`)
- Monitoring system resources
- Cloud example: troubleshooting CPU spikes on EC2

### 5. Permissions and Users
- File permissions and ownership (`chmod`, `chown`, `umask`)
- Groups and users (`useradd`, `usermod`, `sudoers`)
- Understanding `root` privileges
- Cloud example: securing access to cloud VMs

### 6. System Administration
- Managing services (`systemctl`, `journalctl`)
- Network basics (`ping`, `ip`, `ss`, `netstat`)
- SSH configuration and security
- Firewall rules (`ufw`, `iptables`)
- Log monitoring (`/var/log/`)
- Example: securing a Linux host used for ML or cloud workloads
